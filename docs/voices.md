# TTS voice codes

`!tts <message>` speaks in a **random** voice. To pick a specific voice, append its
code to the command with no space, e.g. `!ttsb hello` uses Bella.

Pin this list (or the codes you use most) so chat can find them. Codes are configurable
in the bot; these are the defaults.
